Créer deux bases de données : 
ancestra_other
ancestra_static

- Exécuter le fichier "ancestra_other_full.sql" dans la base de données "ancestra_other".
- Exécuter le fichier "ancestra_static_full.sql" dans la base de données "ancestra_static".

Aller dans le répertoire Patchs et effectuer dans l'ordre croissant les patchs en fonction de leur attribuent.
Ex : Patch revYY - Commentaire _XXXX.sql
YY correspond au numéro du patch.
XXXX correspond a la base de données ou doit être appliquer le patch.

/!\ Attention, les révisions des patchs ne correspondent pas aux révisions de la version du core.
