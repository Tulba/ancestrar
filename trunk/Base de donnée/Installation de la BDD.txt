Créer deux bases de données : 
ancestra_other
ancestra_static

*Comment cela fonctionne ?
Patch revYY - Commentaire _XXXX.sql
YY correspond au numéro du patch.
XXXX correspond a la base de données ou doit être appliquer le patch.


*Si vous exécuter les patchs :
- "ancestra_other_old.sql" dans la base de données "ancestra_other".
- "ancestra_static_old.sql" dans la base de données "ancestra_static".
Aller dans le répertoire Patchs et executer dans l'ordre croissant les patchs en fonction de leur attribuent.

*Si vous exécuter les patchs :
- "ancestra_other_Patched94.sql" dans la base de données "ancestra_other".
- "ancestra_static_Patched94.sql" dans la base de données "ancestra_static".
Aller dans le répertoire Patchs et executer dans l'ordre croissant les patchs supérieur au numéro 94.


/!\ Attention, les révisions des patchs ne correspondent pas aux révisions de la version du core.
